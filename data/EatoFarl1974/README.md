# EatoFarl1974

## What this study folder stores
This folder stores the Eaton and Farley growth curve.

The exposed dataset id is:
- `tL_EatoFarl1974`

## Files and transformation
`raw/tL_EatoFarl1974_raw.csv` stores age and standard length in mm.
`derived/tL_EatoFarl1974.csv` stores the same curve with length converted
to cm for calibration use.
