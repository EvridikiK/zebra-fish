# ValKwa2022

## What this study folder stores
This folder stores the Valentine and Kwasek feeding trial measurements.

The exposed dataset ids are:
- `tWw_ValKwa2022`
- `tL_ValKwa2022`
- `tJX_ValKwa2022`

## Files and columns
`raw/data_ValKwa2022.csv` and `derived/data_ValKwa2022.csv` contain the
same columns and values:
- `age_d`
- `wet_weight_mg`
- `total_length_mm`
- `feed_intake_percent_bw_d`

No transformation script is needed because the source table is already in
the units used by the dataset loader.
