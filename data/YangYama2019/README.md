# YangYama2019

## What this study folder stores
This folder stores the age-length, age-weight, and oxygen-consumption data
from Yang and Yamamoto.

The exposed dataset ids are:
- `tL_YangYama2019`
- `tWw_YangYama2019`
- `tJO_YangYama2019`

## Files and columns
`raw/data_YangYama2019.csv` and `derived/data_YangYama2019.csv` contain:
- `age_d`
- `total_length_cm`
- `wet_weight_g`
- `oxygen_consumption_ug_h`

No transform script is required because the source table is already in the
units used by the dataset loader.
