# Schi2002

## What this study folder stores
This folder stores the larval zebrafish growth curve from Schilling (2002).

The exposed dataset id is:
- `tL_Schi2002`

## Files and transformation
`raw/tL_Schi2002_raw.csv` stores digitized time-length data in days and mm.
`derived/tL_Schi2002.csv` stores the calibration-ready version in days and
cm.

The MATLAB transform converts total length from mm to cm.
